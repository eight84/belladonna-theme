## üåò Installation

1. Go to the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=eight84.belladonna-theme)
2. Click **Install**
3. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and choose `Preferences: Color Theme`
4. Select **Belladonna**
